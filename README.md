# API_foot
API Javascript RESTful dans Node.js.

Projet à l'occasion du programme D-CLIC juste pour construire et déployer cette API sur le site Heroku.

Outils utilisés pour construire l'API :

    NodeJSName
    Express.js - Framework Web rapide, sans opinion et minimaliste pour Node.JS

Paquets installés :

    nvm – Gestionnaire de versions de nœud
    npm - Gestionnaire de paquets de nœuds
    Node.js
    Express installé avec npm (npm install express body-parser morgan).

Comment utiliser :
    https://infinite-journey-36021.herokuapp.com/
